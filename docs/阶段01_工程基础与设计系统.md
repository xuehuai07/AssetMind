# 阶段 01：工程基础与设计系统

## 目标

建立 AssetMind Workbench 的最小可运行工程骨架，为后续 API、检索、Agent 和 UI 打磨提供稳定基础。

## 开发内容

- 初始化 Next.js App Router、React、TypeScript、ESLint、Tailwind CSS 和 PostCSS 配置。
- 建立 `app/layout.tsx`、`app/page.tsx`、`app/globals.css`。
- 定义初版企业工作台视觉方向：克制、清晰、信息密度适中。
- 配置 `npm run dev`、`npm run lint`、`npm run build`。

## 阶段结果

- 工程可安装、可启动、可构建。
- 首页具备基础工作台框架。
- Tailwind 样式和 TypeScript 严格模式生效。
- 后续阶段可在同一工程骨架上继续扩展。

## 验证

- `npm run lint` 通过。
- `npm run build` 通过。

## 风险记录

- 依赖版本较新，后续遇到 Next.js 或 Tailwind 行为差异时，以实际构建结果为准。
